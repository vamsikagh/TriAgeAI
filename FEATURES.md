# ✨ TriageAI - Feature Showcase

## 🎯 Core Features

### 1. AI-Powered Triage Analysis
**Multimodal Input Support:**
- 🎤 **Voice Input** - Web Speech API for hands-free operation
- 📸 **Photo Analysis** - Gemini Vision for visual injury assessment  
- ✍️ **Text Description** - Detailed written injury reports

**AI Assessment Output:**
- Severity classification (RED/YELLOW/GREEN/BLACK)
- Confidence score (0-100%)
- Injury summary and analysis
- Recommended medical actions
- Surgery/blood requirements
- Recommended specialty (Trauma, Ortho, Neuro, etc.)

### 2. Smart Hospital Dispatch
**Intelligent Matching:**
- Specialty-based hospital recommendations
- Real-time bed/ICU/OR availability
- Distance and ETA calculations
- Resource capacity indicators

**Dispatch Features:**
- One-click ambulance assignment
- Turn-by-turn navigation
- Pre-arrival hospital alerts
- Patient data transmission

### 3. Command Dashboard
**Real-Time Monitoring:**
- Active event tracking
- Live patient statistics
- Resource availability overview
- Hospital network status

**Key Metrics:**
- Total patients triaged
- Severity distribution (Red/Yellow/Green/Black)
- Available ambulances
- Hospital bed capacity
- Blood units, ventilators, OR availability

### 4. Resource Management
**Track Critical Resources:**
- 🚑 Ambulances (12 total, 8 available)
- 🛏️ Hospital Beds (50 total, 32 available)
- 👨‍⚕️ Surgeons (8 total, 5 available)
- 🩸 Blood Units (200 total, 145 available)
- 💨 Ventilators (15 total, 11 available)
- 🏥 Operating Rooms (6 total, 4 available)

**Visual Indicators:**
- Color-coded status bars (Red/Yellow/Green)
- Real-time updates as resources are allocated
- Percentage availability display

### 5. Hospital Network Map
**Interactive Features:**
- Google Maps integration
- Hospital location markers
- Capacity indicators
- Specialty information
- Distance and ETA display

**Hospital Data:**
- 5 hospitals in Kochi/Kerala region
- Specialty types (Trauma, Emergency, Multi-Specialty, etc.)
- Bed counts and availability
- Real-time status updates

### 6. Patient Management
**Comprehensive Patient Records:**
- Unique patient IDs (TC-0001, TC-0002, etc.)
- Timestamp of triage
- Severity classification
- Injury details
- Assigned hospital
- Transport status

**Patient List Features:**
- Sortable and filterable
- Severity color coding
- Quick status updates
- Export capabilities
